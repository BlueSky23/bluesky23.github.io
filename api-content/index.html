在传统的编程方法中，都是我们告诉计算机来做什么。通常，我们将一个大问题分解为多个计算机可以轻松执行的规模较小且明确定义的任务，让计算机执行。相比之下，在神经网络中，我们不会告诉计算机如何解决问题。取而代之的是，神经网络将从数据中自主学习，从而找到自己的解决方案。 从观测数据自主学习听起来很有前景。 但是，在2006年之前，除少数特殊问题外，我们还不知道如何训练神经网络以让它能够超越传统的学习方法。在2006年，科学家提出了神经网络中的学习技术，即深度学习技术。如今，深度学习技术已经得到了进一步的发展，使得神经网络在计算机视觉、语音识别和自然语言处理等许多重要问题上都取得了卓越的效果，Google、Microsoft和Facebook等公司也正在大规模部署和使用它们。 本书的目的就是帮助您掌握神经网络的核心概念，也包括用于深度学习的现代化技术和方法。读完本书后，您将可以编写使用神经网络和深度学习解决复杂模式识别问题的代码，并将具备使用神经网络和深度学习来解决实际问题的基础。 以原则为导向的方法 该书的一个目标是对神经网络和深度学习的核心原理有扎实的理解，而不仅是对冗长的概念清单产生朦胧的理解。如果您已经很好地理解了核心原理，则可以快速理解其他相关的新材料。用编程语言类比，可以认为它是掌握一种新语言的核心语法、标准库和数据结构，虽然您可能只“知道”全部语言的一小部分（许多语言都有大量的标准库），但是可以快速、轻松地理解新的库。 我们将通过解决一个具体问题来学习神经网络和深度学习背后的核心原理，即教计算机识别手写数字的问题。这个问题，使用常规的编程方法很难解决，但是，我们只需要用简单的神经网络就能很好地解决它，整个程序只需几十行代码，也无需特殊的库。而且，在整本书中，我们将通过多次改进这个程序，逐步介绍有关神经网络和深度学习的越来越多的核心思想。
